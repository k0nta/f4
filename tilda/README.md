# Импорт в Tilda

В Tilda этот лендинг лучше вставлять как кастомный HTML-блок.

## Вариант 1 - легче и правильнее

1. Загрузите файлы из папки `tilda/assets/` в Tilda или на любой CDN/хостинг.
2. Откройте `tilda/f4-tilda-custom-code.html`.
3. Замените `__ASSET_BASE_URL__` на URL папки с загруженными ассетами без слеша в конце.
   Пример: `https://static.tildacdn.com/your-folder`
4. В Tilda добавьте блок `T123 HTML-код`.
5. Вставьте весь код из `f4-tilda-custom-code.html`.

## Вариант 2 - одним куском

Файл `tilda/f4-tilda-inline.html` уже содержит изображения внутри HTML.
Его можно вставить в `T123 HTML-код` без отдельной загрузки ассетов.

Минус: файл весит около 5 МБ, поэтому в редакторе Tilda он может открываться тяжелее.

## Что важно

- Сайт в Tilda не станет редактируемыми стандартными блоками. Он будет работать как один кастомный HTML-блок.
- Стили изолированы внутри `#f4-tilda-site`, чтобы не ломать остальную страницу.
- Внешняя ссылка бронирования сейчас стоит `https://f4club.ru/`; ее можно заменить в HTML.
